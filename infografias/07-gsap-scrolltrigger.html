GSAP ScrollTrigger

El plugin de GSAP que conecta animaciones con el scroll del usuario

Viewport y trigger
Viewport
start: "top center"
trigger
end: "bottom center"
↑ scroll ↓
Tres superpoderes
⚙

Scrub

Vincula el progreso de la animacion directamente al scroll

📌

Pin

Fija un elemento en pantalla mientras el scroll continua

☰

Timeline

Secuencia multiples animaciones en una sola linea temporal

Codigo esencial
Scrub + Pin JS
gsap.to(".hero", {
  opacity: 0,
  scrollTrigger: {
    trigger: ".hero",
    scrub: true,
    pin: true,
  }
});
Toggle Actions JS
scrollTrigger: {
  trigger: ".card",
  start: "top 80%",
  end: "top 20%",
  toggleActions:
    "play none none reverse"
  // onEnter onLeave
  // onEnterBack onLeaveBack
}
