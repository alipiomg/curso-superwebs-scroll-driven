Flujo de trabajo
1

Definir producto

Caracteristicas clave, materiales, forma, color

2

Crear prompt base

Describir el producto con todos sus atributos visuales

3

Variar angulos

Generar multiples vistas: frontal, lateral, cenital, 3/4

4

Verificar consistencia

Comparar colores, proporciones y detalles entre tomas

Consejos clave

⚡ Tips para consistencia

  • Usa seed fijo cuando la herramienta lo permita para mantener coherencia entre generaciones
  • Incluye descriptores de material en cada prompt: matte, glossy, brushed metal, etc.
  • Mantener el mismo estilo de iluminacion en todas las variantes del producto
  • Generar en alta resolucion (minimo 1024x1024) para tener margen de recorte
  • Guardar los prompts base en un documento para reutilizar y iterar
  • Verificar que el color del producto no cambie entre angulos diferentes
