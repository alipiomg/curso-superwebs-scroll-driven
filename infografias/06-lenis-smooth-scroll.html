Lenis Smooth Scroll

Scroll suave de alto rendimiento con interpolacion lineal para experiencias premium

Nativo vs Lenis

📈 Scroll nativo

pos tiempo

Movimiento escalonado: saltos bruscos entre posiciones. Cada tick de rueda salta un bloque fijo de pixeles.

VS

📈 Lenis scroll

pos tiempo

Curva suave y continua: interpolacion frame a frame. El scroll se siente organico y controlado.

Formula Lerp

Linear Interpolation (Lerp)

lerp(a, b, t) = a + (b - a) * t
t = 0.1 significa que cada frame recorre el 10% de la distancia restante
a
Posicion actual
Donde esta el scroll ahora mismo en este frame
b
Posicion objetivo
Donde quiere llegar el scroll segun el input del usuario
t
Factor de suavizado
Velocidad de convergencia. Bajo = mas suave, alto = mas rapido
// En cada frame del requestAnimationFrame
const current = lerp(currentScroll, targetScroll, 0.1);
// Frame 1: 0 + (1000 - 0) * 0.1 = 100
// Frame 2: 100 + (1000 - 100) * 0.1 = 190
// Frame 3: 190 + (1000 - 190) * 0.1 = 271
Flujo de integracion
1

Instalar

Agregar Lenis al proyecto via npm o CDN

npm i lenis
2

Inicializar

Crear instancia con configuracion de lerp y duracion

new Lenis({ lerp: 0.1 })
3

Conectar con rAF

Vincular al loop de animacion del navegador

lenis.raf(time)
